 # Cyber Security Project

## Project Title
Integrate Zscaler with Identity Providers (IdPs) such as Azure AD, Okta and Active Directory for seamless authentication and authorization.

## Project Overview
This project focuses on identifying web application vulnerabilities and understanding modern security architecture using Zscaler and Identity Providers. The project also demonstrates vulnerability scanning and Zero Trust security concepts.

## Project Stages
1. Web Application Vulnerability Analysis
2. Vulnerability Scanning using Nessus
3. Zscaler Integration with Identity Providers

## Tools Used
- Browser Developer Tools
- Nessus Vulnerability Scanner
- Zscaler Security Platform
- OWASP Security Concepts
